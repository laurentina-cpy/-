# 城小二管理系统 - 快速启动指南

## 一、安装与运行

### 1. 安装依赖
```bash
cd chengxiaoer-admin
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

默认访问地址: http://localhost:5173

### 3. 登录系统

#### 使用模拟数据登录（默认已启用）
- 用户名: `admin`
- 密码: `任意密码`

## 二、系统功能概览

### 📊 仪表板
- 查看系统统计数据（用户、商品、订单、销售额）
- 查看销售趋势图表
- 查看热门商品排行

### 🛍️ 商品管理
- **查看**: 支持按商品名称、分类、上架状态搜索
- **新增**: 填写商品名称、分类、价格、库存、描述
- **编辑**: 修改商品信息
- **删除**: 删除单个商品或批量删除
- **状态**: 上架/下架商品

### 👥 用户管理
- **查看**: 支持按用户名/手机号、状态搜索
- **操作**: 启用/禁用用户账号
- **统计**: 查看用户订单数和消费金额

### 📂 分类管理
- **查看**: 查看所有商品分类及商品数量
- **新增**: 创建新分类（名称、描述、排序）
- **编辑**: 修改分类信息
- **删除**: 删除不需要的分类

### 🏭 供应商管理
- **查看**: 支持按名称、联系人搜索
- **新增**: 添加新供应商（名称、联系人、电话、地址）
- **编辑**: 修改供应商信息
- **删除**: 删除供应商
- **状态**: 启用/禁用供应商

### 🚚 配送员管理
- **查看**: 支持按姓名/手机号、状态搜索
- **状态**: 空闲/忙碌/离线
- **新增**: 添加新配送员（姓名、手机、身份证、车牌）
- **编辑**: 修改配送员信息
- **统计**: 查看配送单数和配送费收入

### 📋 订单管理
- **查看**: 支持按订单号、手机号、状态、日期范围搜索
- **详情**: 查看订单完整信息（商品、地址、金额）
- **分配**: 将订单分配给可用配送员
- **完成**: 标记订单为已完成
- **取消**: 取消待支付或待配送的订单
- **导出**: 导出订单数据为Excel

## 三、数据模拟

系统已内置模拟数据，无需后端即可测试所有功能。

### 修改真实API连接

如需连接后端API，修改 `src/utils/request.js`:

```javascript
const ENABLE_MOCK = false  // 改为 false

const service = axios.create({
  baseURL: 'http://your-api-server.com/api',  // 修改为你的API地址
  timeout: 10000
})
```

## 四、常用操作说明

### 1. 搜索过滤
- 所有列表页面都支持搜索功能
- 填写搜索条件后点击"搜索"按钮
- 点击"重置"按钮清空搜索条件

### 2. 分页
- 默认每页显示10条数据
- 可切换为 20/50/100 条每页
- 支持页码跳转

### 3. 批量操作
- 商品管理支持批量删除
- 勾选多个商品后点击"批量删除"按钮

### 4. 状态切换
- 商品、用户、供应商的状态使用开关组件
- 点击开关立即更新状态

### 5. 订单流程
```
待支付 → 待配送 → 配送中 → 已完成
                ↓
              已取消
```

## 七、常见问题

### Q: 登录后提示401错误？
A: 检查 `ENABLE_MOCK` 是否为 `true`，或确认后端API是否正常运行

### Q: 列表数据不显示？
A: 检查浏览器控制台是否有网络错误，确认API地址正确

### Q: 图表不显示？
A: 确认已安装 echarts 依赖，检查数据格式是否正确

### Q: 导出功能报错？
A: 导出需要后端支持，确认后端已实现导出接口

## 八、开发建议
